# SmartProcureRadar V2.0

# 企业 IT 采购运营与决策支持平台升级设计文档

## 项目定位

SmartProcureRadar 是一个面向企业 IT 采购场景的数字化采购管理模拟平台。

目标： 模拟真实企业 IT 采购部门工作流程：

PR采购申请 → 审批 → 供应商管理 → RFQ询价 → 报价分析 → 采购决策 → PO订单
→ 合同管理 → 验收 → 付款 → 数据分析

该项目用于展示软件技术专业学生对企业 IT
采购流程、数据分析、云服务以及数字化管理工具的理解。

# 八、简历描述

项目名称：

SmartProcureRadar 企业IT采购运营管理平台

项目描述：

独立设计并实现企业IT采购运营管理平台，模拟PR申请、供应商管理、RFQ询价、报价分析、采购订单、合同、验收及付款完整流程。

项目亮点：

-   构建供应商多维评分模型，实现价格、交付、服务、质量综合分析。
-   设计云服务采购比较模块，对腾讯云、阿里云、AWS资源成本进行分析。
-   实现采购流程状态管理，模拟企业采购审批与订单执行过程。
-   基于AI助手实现采购建议生成和风险分析。
-   支持Excel数据导出和云端部署。

------------------------------------------------------------------------
